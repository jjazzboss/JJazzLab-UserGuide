# Contribuer à la documentation

Ce site est hébergé par GitBook, qui permet aux utilisateurs de mettre à jour la documentation, comme dans un Wiki. Effectuer des modifications est assez simple car cela se fait directement dans le navigateur internet via un éditeur WYSIWYG.

Si vous souhaitez contribuer, envoyez-moi simplement un e-mail \([ici](https://www.jjazzlab.com/en/contact/)\) et expliquez-moi brièvement ce que vous souhaitez faire afin que je puisse vous accorder les droits d'accès en écriture au GitBook de JJazzLab.

Si vous êtes un utilisateur GitHub, vous pouvez également faire une pull-request directement sur le [dépôt JJazzLabDoc](https://github.com/jjazzboss/JJazzLabDoc).
