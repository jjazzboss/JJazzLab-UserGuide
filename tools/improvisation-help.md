# アドリブのヒント

JJazzLabは、ハル・クルーク(Hal Crook)著『How to improvise』からアドリブ勉強に役立つエクササイズをご紹介しています。

JJazzLabは、アクティブにしている曲に合ったエクササイズの演奏のガイドを生成します。
