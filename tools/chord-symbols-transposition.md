# Transposition des chord symbols

Si vous jouez d'un instrument transpositeur (sax alto, sax ténor, flûte alto, etc.), utilisez cette fonctionnalité pour que les chord symbols s'affichent transposés. La musique générée réellement n'est pas modifiée.

Ces chord symbols transposés apparaissent en _italique_.
