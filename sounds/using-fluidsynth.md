# Using FluidSynth

JJazzLab intègre une instance FluidSynth **prête à l'emploi** et **configurée de manière optimale**.&#x20;

FluidSynth devrait être activé par défaut lors du premier lancement de JJazzLab sous Windows.&#x20;

Sous Linux et MacOS, il est nécessaire que FluidSynth ait été préalablement installé sur le système.&#x20;

{% hint style="danger" %}
Si FluidSynth apparaît désactivé, cela signifie qu'un problème est survenu lors du chargement de FluidSynth. Consultez les [**instructions d'installation**](../installation.md)**.**
{% endhint %}
