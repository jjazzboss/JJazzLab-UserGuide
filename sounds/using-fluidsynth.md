# FluidSynthの使用

これはJJazzLab用に**最適に設定された****すぐに使える**シンセです。FluidSynthは、JJazzLabを初めて起動した際にデフォルトで有効になっているはずです。

{% hint style="danger" %}
FluidSynthが無効になっている場合、FluidSynthの読み込み中に問題が発生したことを意味します。[インストール方法](../installation.md)をチェックしてください。
{% endhint %}
