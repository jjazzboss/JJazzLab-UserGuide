# コマンド

伴奏トラックの再生は、下のツールバーでコントロールできます。

![](../.gitbook/assets/MusicControlToolbarText.png)

**Metronome**と**Precount mode**は、**ツール/設定**の**Click**で指定します。

### キーボードショートカット

{% hint style="warning" %}
ショートカットは、WindowsとLinux用です。Macの場合は、一部変更が必要でしょう。
{% endhint %}

|                                                 |                                                     |
| ----------------------------------------------- | --------------------------------------------------- |
| <mark style="background-color:blue;">キー</mark> | <mark style="background-color:blue;">コマンド</mark>|
| space                                           | 再生 / 一時停止 / 再開                               |
| shift-space                                     | 停止                                                |
| ctrl-space                                      | 選択した小節／ソングパートから再生                     |
| ctrl-shift-space                                | 選択した小節／ソングパートのみ再生                     |
| F1                                              | 前のソングパートに戻って再生                          |
| F2                                              | 次のソングパートに進んで再生                          |
| F3                                              | ソングの頭から再生 (リスタート)                       |
| J or -                                          | テンポを遅くする                                     |
| K or +                                          | テンポを速くする                                     |
| L                                               | ループモードの切り替え                                |
