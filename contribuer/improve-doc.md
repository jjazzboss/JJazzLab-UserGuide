# Améliorer la documentation

## Tout utilisateur peut modifier le document en ligne

Ce site est hébergé sur la plateforme GitBook, **qui permet aux utilisateurs de mettre à jour la documentation**, comme dans un Wiki. Faire des modifications est très facile car cela se fait directement dans le navigateur Internet, sans rien à installer.

Si vous souhaitez apporter des modifications, envoyez-moi un email ([ici](https://www.jjazzlab.com/en/contact/)) et je vous répondrai avec un lien pour vous accorder des droits d'accès en écriture au GitBook JJazzLab.&#x20;

{% hint style="info" %}
Si vous êtes un utilisateur GitHub, vous pouvez également utiliser le bouton **Edit on GitHub** en haut à droite de chaque page, ou faire une pull-request directement sur le [dépôt JJazzLab-UserGuide](https://github.com/jjazzboss/JJazzLab-UserGuide).
{% endhint %}

## Traduire le document en ligne

Vous pouvez également aider en corrigeant ou en ajoutant de nouvelles traductions. Actuellement, il y a 2 langues, l'anglais et le japonais. Si votre langue n'existe pas encore, contactez-moi ([ici](https://www.jjazzlab.com/en/contact/)) pour que j'ajoute une « variante GitBook » pour cette langue.
