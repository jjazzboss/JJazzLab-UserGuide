# Améliorer la documentation

## Tout utilisateur peut modifier la documentation en ligne

Ce site est hébergé sur la plateforme GitBook, **qui permet aux utilisateurs de mettre à jour la documentation**, comme dans un Wiki. Effectuer des modifications est très simple car cela se fait directement dans le navigateur internet, sans rien à installer.

Si vous souhaitez apporter des modifications, envoyez-moi simplement un e-mail ([ici](https://www.jjazzlab.com/en/contact/)) et je vous répondrai avec un lien pour vous accorder les droits d'accès en écriture au GitBook de JJazzLab.&#x20;

{% hint style="info" %}
Si vous êtes un utilisateur GitHub, vous pouvez également utiliser le bouton **Edit on GitHub** en haut à droite de chaque page, ou faire une pull-request directement sur le [dépôt JJazzLab-UserGuide](https://github.com/jjazzboss/JJazzLab-UserGuide).
{% endhint %}

## Traduire la documentation en ligne

Vous pouvez également aider en corrigeant ou en ajoutant de nouvelles traductions. Il existe actuellement 2 langues, l'anglais et le japonais. Si votre langue n'existe pas encore, contactez-moi ([ici](https://www.jjazzlab.com/en/contact/)) afin que j'ajoute une « variante GitBook » pour cette langue.
