# Améliorer la documentation

Ce site est hébergé sur la plateforme GitBook, qui permet aux utilisateurs de mettre à jour la documentation, comme dans un Wiki. Effectuer des modifications est assez simple car cela se fait directement dans le navigateur internet avec un éditeur simple.

Si vous souhaitez apporter des modifications, envoyez-moi simplement un e-mail \([ici](https://www.jjazzlab.com/en/contact/)\) et expliquez-moi brièvement ce que vous souhaitez faire afin que je puisse vous accorder les droits d'accès en écriture au GitBook de JJazzLab.

Si vous êtes un utilisateur GitHub, vous pouvez également utiliser le bouton **Edit on GitHub** en haut à droite de chaque page, ou faire une pull-request directement sur le [dépôt JJazzLabDoc](https://github.com/jjazzboss/JJazzLabDoc).
