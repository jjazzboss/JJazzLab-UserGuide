## Anleitung für Windows&#x20;

Laden Sie das Setup Programm runter und starten Sie es.&#x20;

{% hint style="info" %}
Falls Sie keine Administratorrechte haben, wählen Sie während der Installation "Nur für mich installieren".
{% endhint %}

Wir empfehlen es, JJazzLab mit [JJazzLab SoundFont](/broken/pages/-MQEBGOlE-e0AupJEQ_k) zu benutzen.

:arrow\_right: Falls Sie eine schrittweise Installationsanleitung brauchen, schauen Sie bitte diese [Installation Videoanleitung](video-tutorials.md#installation-windows-only) (englisch).

{% hint style="warning" %}
Falls Sie eine **Windows Smart Screen** Warnung bekommen:

<img src=".gitbook/assets/Win10SmartScreen.png" alt="" data-size="original">&#x20;

Windows Smart Screen blockiert das Programm NICHT weil es ein schädliches Programm ist (ist es nicht!), sondern nur, weil JJazzLab neu ist und Windows Sicherheitsservers nicht über genug Statistik verfügt, um seine Sicherheit zu evaluieren.

Sobald genug Nutzer das Programm erfolgreich runtergeladen und installiert haben, wird Windows Smart Screen das Programm nicht länger blockieren.

Sie können eine ausführlichere Erklärung hierzu in diesem[ Artikel](https://www.digitalcitizen.life/what-smartscreen-filter-how-does-it-work) finden.
{% endhint %}



## Linux instructions

Unzip the package.&#x20;

Start JJazzLab using **bin/jjazzlab** in the installation directory.&#x20;

We recommend to use JJazzLab with the [JJazzLab SoundFont](/broken/pages/-MQEBGOlE-e0AupJEQ_k).

{% hint style="success" %}
If you use Arch or Manjaro Linux, there is JJazzLab installation package available : [https://aur.archlinux.org/packages/jjazzlab-x-bin/](https://aur.archlinux.org/packages/jjazzlab-x-bin/)    (thanks to jypma!)&#x20;
{% endhint %}

## MacOS instructions

Unzip the package and open it to start JJazzLab.&#x20;

{% hint style="warning" %}
If you get a **security alert**

Using the **Finder**, select the JJazzLab package, **ctrl-click menu**, **Open**, this will give you the choice to open the application in spite of the security alert.
{% endhint %}

We recommend to use JJazzLab with the [JJazzLab SoundFont](/broken/pages/-MQEBGOlE-e0AupJEQ_k).

{% hint style="warning" %}
There is a known bug on MacOS: sometimes, after the computer wakes up from sleep mode, the audio can lag 2 or 3 seconds behind. This is due to a JDK problem, JJazzLab can't do anything.
{% endhint %}
