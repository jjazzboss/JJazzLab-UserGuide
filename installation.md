## Instrucciones para Windows

Descarga y corre el programa de instalación.

{% hint style="info" %}
Si no tienes derecho de administrador en tu PC puedes elegir "instalar solamente para mí" en el menú de instalación.
{% endhint %}

Recomendamos encarecidamente usar JJazzLab con las fuentes/archivos de Sonido [JJazzLab SoundFont](/broken/pages/-MQEBGOlE-e0AupJEQ_k) (mejor calidad).

{% hint style="warning" %}
Si recibes una ventana de alerta de **Windows Smart Screen**

<img src=".gitbook/assets/Win10SmartScreen.png" alt="" data-size="original">&#x20;

Windows Smart Screen bloquea el programa NO PORQUE SEA UN MALWARE (No lo es!) sino porque JJazzLab es un software nuevo. Al ser nuevo los servidores de seguridad de Windows no tienen las suficientes estadísticas disponibles como para evaluar su "reputación de seguridad" y por ende lo bloquean.&#x20;

Una vez que una cantidad suficiente de usuarios de Windows hayan descargado e instalado JJazzLab, Windows Smart Screen no bloqueará más la instalación.

Puedes encontrar las explicaciones en este artículo (inglés) => [good article](https://www.digitalcitizen.life/what-smartscreen-filter-how-does-it-work).
{% endhint %}

## Instrucciones para Linux&#x20;

Descarga y descomprime el paquete.&#x20;

Para ejecutar JJazzLab usa el archivo jjazzlab dentro de la carpeta "bin". Se encuentra en el directorio de instalación.

Recomendamos encarecidamente usar JJazzLab con las fuentes/archivos de Sonido [JJazzLab SoundFont](/broken/pages/-MQEBGOlE-e0AupJEQ_k) (mejor calidad).

{% hint style="success" %}
Si utilizas la distribución Arch o Manjaro Linux hay un paquete disponible para instalar JJazzLab:  [https://aur.archlinux.org/packages/jjazzlab-x-bin/](https://aur.archlinux.org/packages/jjazzlab-x-bin/)    (Muchas gracias jypma!)&#x20;
{% endhint %}

## Instrucciones para MacOS&#x20;

Descarga y descomprime el paquete para ejecutar JJazzLab.&#x20;

{% hint style="warning" %}
Si aparece una **alerta de seguridad**

Usando **Finder**, selecciona el paquete JJazzLab, usando **ctrl-click menu**, **Abrir**, podrás elegir ejecutar la aplicación sin tener en cuenta el alerta de seguridad.
{% endhint %}

Recomendamos encarecidamente usar JJazzLab con las fuentes/archivos de Sonido [JJazzLab SoundFont](/broken/pages/-MQEBGOlE-e0AupJEQ_k) (mejor calidad).

{% hint style="warning" %}
Hay un error conocido en MacOS: a veces, luego de que la PC hiberna el audio puede tener un retardo de 2 o 3 segundos. Esto se debe a un problema de JDK, JJazzLab no puede hacer nada
{% endhint %}
