# Overview

<figure><img src="../.gitbook/assets/EditorsWithText.png" alt=""><figcaption><p>Le song structure editor vous permet de définir comment les sections composent le song final</p></figcaption></figure>

### Utilisez le **Lead Sheet Editor** pour

* Ajouter des [chord symbols](chord-lead-sheet.md#chord-symbols), ex. "Cm6", "Ab7"
* Ajouter des [sections](chord-lead-sheet.md#sections-input), ex. "A", "B" en 3/4, "verse"
* Déplacer et modifier des chord symbols pour ajuster les accents rythmiques, l'[interprétation](chord-lead-sheet.md#interpretation) ou l'[harmonie](chord-lead-sheet.md#harmony)
* Ajouter des [bar annotations](chord-lead-sheet.md#bar-annotations-lyrics) ou des paroles (optionnel)

Visitez la page [lead sheet](chord-lead-sheet.md) pour plus d'informations.

### Utilisez le **Song Structure Editor** pour

* Définir l'ordre des sections à l'aide de [song parts](song-structure.md#song-parts), ex. "AABA", "verse verse chorus verse", ...
* Sélectionner les [rhythms](song-structure.md#change-rhythm) (styles musicaux) à utiliser dans le song
* Introduire des dynamiques et des variations grâce aux [rhythm parameters](song-structure.md#rhythm-parameters) sur chaque song part

Visitez la page [song structure](song-structure.md) pour plus d'informations.

### Utilisez le **Mix Console** pour

* Définir l'instrument de chaque piste
* Ajuster le volume, le panoramique, la réverbération, le chorus, la sourdine, la transposition, etc. de chaque piste
* Ajouter des [user tracks](mix-console.md#user-tracks)

Visitez la page [mix console](mix-console.md) pour plus d'informations.

### Utilisez le Note Editor pour

* Modifier un [user track](mix-console.md#user-tracks)
* [Personnaliser](song-structure.md#custom-phrase) une phrase d'instrument d'un song part

Visitez la page [note editor](overview.md#note-editor) pour plus d'informations.

### Détacher / rattacher les éditeurs

Pour détacher un éditeur, sélectionnez **Float** dans le menu contextuel de son onglet.

L'image ci-dessous montre le menu contextuel de l'onglet du [Song part editor](song-structure.md#song-part-editor).

<figure><img src="../.gitbook/assets/WindowUndockPopupMenu.png" alt=""><figcaption><p>Sur Windows ou Linux, le menu contextuel de l'onglet de l'éditeur apparaît lorsque vous faites un clic droit dessus</p></figcaption></figure>

<figure><img src="../.gitbook/assets/WindowUndocked2.png" alt=""><figcaption><p>L'éditeur flottant peut être déplacé et redimensionné indépendamment de la fenêtre principale</p></figcaption></figure>

Pour rattacher l'éditeur flottant, sélectionnez **Dock** dans le même menu contextuel d'onglet.

Utilisez **Reset Windows** dans le menu principal Fenêtre pour rattacher tous les éditeurs.

{% hint style="warning" %}
Le lead sheet et le song structure editors ne peuvent pas être détachés/rattachés
{% endhint %}
