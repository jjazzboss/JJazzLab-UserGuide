# Song memo

<figure><img src="../.gitbook/assets/SongMemoMenu.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/SongMemo.png" alt=""><figcaption></figcaption></figure>

Utilisez le bouton **Insert link...** pour insérer un lien vers un fichier sur l'ordinateur ou une page web.

Le bouton **Open links** ouvre tous les liens du memo. Les liens sont automatiquement ouverts lorsque vous ouvrez le fichier song.

{% hint style="danger" %}
Pour des raisons de sécurité, vous ne pouvez insérer que 4 liens au maximum avec les extensions suivantes :

* "pdf", "txt", "md", "rtf"
* &#x20;"jpg", "jpeg", "png", "gif", "webp", "svg", "bmp"
* "mp3", "wav", "flac", "ogg", "m4a", "aac", "wma", "mp4", "mov", "avi", "mkv", "webm"
* "mid", "midi"
* "mscz", "mscx", "xml", "musicxml", "gp", "gpx", "gp5", "ly", "sib"
* "doc", "docx", "xls", "xlsx", "odt", "ods"
{% endhint %}
