# Inspecteur d'accords

Utilisez l'inspecteur d'accords (menu **Tools/Chord inspector**) pour voir les notes du **chord symbol sélectionné**, avec différentes visualisations, comme indiqué dans l'image ci-dessous.

## Clavier piano

![](<../.gitbook/assets/2021-12-13 20_34_45-JJazzLab.png>)

S'il existe une **gamme harmonique** définie pour le chord symbol sélectionné (voir l'[éditeur de chord symbol](../editeurs/chord-lead-sheet.md#harmony)), les notes de la gamme sont affichées (exemple ci-dessous).

![](<../.gitbook/assets/2021-12-13 20_36_34-JJazzLab.png>)

## Diagrammes de guitare

Les points rouges représentent les notes toniques (ex. G pour Gm69).

![Diagrammes les plus courants pour l'accord G69](<../.gitbook/assets/2021-12-13 20_35_23-JJazzLab.png>)

En cliquant sur le bouton, vous pouvez basculer entre différents types de positions : _les plus courants, voix ouverte, voix serrée, inversions_.

![Diagrammes en voix ouverte pour l'accord Gm69](<../.gitbook/assets/2021-12-13 20_35_59-JJazzLab.png>)
