# YamJJazzリズムエンジン

JJazzLabは **YamJJazz** リズム生成エンジンを組み込んでいます。このエンジンは[ヤマハスタイル](yamaha-styles.md)を読み込み、既存のヤマハスタイルファイルにさらなるバリエーションを追加する新しい[拡張ヤマハスタイル](extended-yamaha-styles.md)フォーマットを導入します。
